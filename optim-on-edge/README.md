# OPTIM-ON Edge

> **Warning:** This is the **development/edge** channel of OPTIM-ON. Builds are created on every commit to `main` and may be unstable. For production use, install the [stable version](../optim-on/README.md).

OPTIM-ON Edge provides early access to the latest features and fixes by DOMOPTI.

## Installation

1. Add this repository to your Home Assistant addon store:
   `https://github.com/DOMOPTI/HA-addons`
2. Find **OPTIM-ON Edge** in the addon store and install it.
3. Start the addon.
