# DOMOPTI HA Addons

Home Assistant Addons Repository by DOMOPTI S.R.L.S.

## Installation

Add this repository to your Home Assistant addon store by navigating to **Settings → Add-ons → Add-on Store → Menu (⋮) → Repositories** and adding:

```
https://github.com/DOMOPTI/HA-addons
```
