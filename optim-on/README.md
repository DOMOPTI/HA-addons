# OPTIM-ON

OPTIM-ON is an energy optimization system by DOMOPTI for Home Assistant.

## Installation

1. Add this repository to your Home Assistant addon store:
   `https://github.com/DOMOPTI/HA-addons`
2. Find **OPTIM-ON** in the addon store and install it.
3. Start the addon.

## Configuration

No configuration options are required by default.
