# DOMOPTI HA Addons

Home Assistant Addons Repository by DOMOPTI.

## Installation

Add this repository to your Home Assistant addon store by clicking the button below or by navigating to **Settings → Add-ons → Add-on Store → Menu (⋮) → Repositories** and adding:

```
https://github.com/DOMOPTI/HA-addons
```
